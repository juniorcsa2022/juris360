# Dashboard (Visão Geral)

## Objetivo
Fornecer um resumo rápido da situação do escritório assim que o usuário faz login.

### Widgets Principais (Previstos)

1. **Resumo Financeiro**
   - Total a Receber (Mês atual).
   - Total a Pagar (Mês atual).
   - Saldo em Caixa.
   - Gráfico de fluxo de caixa (Receitas x Despesas).

2. **Prazos e Audiências (Próximos 7 dias)**
   - Lista de compromissos urgentes.
   - Contagem de prazos fatais.

3. **Novas Publicações**
   - Contador de publicações não lidas.
   - Atalho para tratar publicações.

4. **Processos Recentes**
   - Lista dos últimos processos movimentados ou cadastrados.

### Navegação (Sidebar)
Menu lateral persistente com acesso aos módulos:
- Dashboard
- Processos
- Agenda
- Financeiro
- Clientes
- Documentos
- Configurações
